## :notebook: About this Project

It is a project to practice. A bank can create a user account, deposit, withdraw and make transactions for other accounts


### Installing



**Installing dependencies**

```bash
$ yarn
```
or
```bash
$ npm install
```

### Running

> Localhost Web
```bash
$ yarn start
```
or
```bash
$ npm start
```
